Model

A generative model for complex and hierarchical data.

class gallimimus.MetaLearner(codec_in)

Bases: object

The MetaLearner is the complete generative model. The type of the data it generates is parametrized by the provided Codec.

Parameters:

codec_in (Codec) – The codec corresponding to the data type generated by the model.

Sample a batch of chosen size from the model from the distribution represented by the provided parameters.

loss_and_grad(params, xs)

For a batch of observations xs, compute the average loss (the NLL of x in the distribution represented by params) and its associated gradient.

Parameters:

  • params (flax.core.scope.VariableDict) – A Pytree of parameters.

  • xs (Observation) – A batch of observations, stacked on the first dimension of the leaves.

Returns:

A tuple containing the average loss of the batch (an array of shape ()) and its gradient with respect to the parameters (a Pytree of the same shape as params).

Return type:

Tuple[Array, Any]

loss_and_per_example_grad(params, xs)

For a batch of observations xs, compute the average loss and per-instance gradients (per-instance gradients are required for DP-SGD, so that they can be clipped).

Parameters:

  • params (flax.core.scope.VariableDict) – A Pytree of parameters.

  • xs (Observation) – A batch of observations, stacked on the first dimension of the leaves.

Returns:

A tuple containing the average loss of the batch (an array of shape ()) and the per-instance gradient with respect to the parameters (a stack of Pytrees of the same shape as params, stacked on the first dimension of the leaves).

Return type:

Tuple[Array, Any]
